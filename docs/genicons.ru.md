# Generating .3db icons

Чтобы создать .3db иконку, выберите **Tools→Generate Icon**, затем в появившемся окне укажите файл изображения. Изображение должно иметь квадратные размеры и быть степенью двойки (например: 32x32, 128x128, 256x256).

![New Icon Dialog](assets/newicondlg.png)

- **Compress** — включите, чтобы сжать иконку с использованием DXT5. Это уменьшает размер файла на диске и в VRAM, но может привести к небольшим артефактам сжатия. Если включаете эту опцию, проверьте иконку в Model Viewer или в игре, чтобы убедиться, что качество вас устраивает.
- **Icon Name** — задаёт базовое имя для текстуры и узлов материала иконки.
